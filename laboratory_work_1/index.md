# Задание 1 — UDP клиент/сервер

**Описание:** клиент отправляет серверу сообщение `Hello, server`.  
Сервер выводит сообщение в консоль и отвечает `Hello, client`.

### Файлы
- `task1_udp/server.py`
- `task1_udp/client.py`

### Запуск
Сначала сервер (окно 1):
```bash
python task1_udp/server.py
```

Затем клиент (окно 2):
```bash
python task1_udp/client.py
```

### Результат
- на сервере отображается: `Client said: Hello, server`
- на клиенте отображается: `Server said: Hello, client`

# Задание 2 — TCP: математическая операция (вариант 1)

**Вариант 1:** теорема Пифагора. Клиент вводит катеты `a` и `b`, сервер возвращает гипотенузу:  
`c = sqrt(a^2 + b^2)`.

### Файлы
- `task2_tcp_math/server.py`
- `task2_tcp_math/client.py`

### Запуск
Сервер (окно 1):
```bash
python task2_tcp_math/server.py
```

Клиент (окно 2):
```bash
python task2_tcp_math/client.py
```

### Результат
Клиент выводит вычисленное значение `c` (гипотенузу).

# Задание 3 — HTTP-сервер: отдача страницы из файла

**Описание:** клиент (браузер) подключается к серверу и получает HTTP-ответ,  
который содержит HTML-страницу, подгруженную сервером из файла `index.html`.

### Файлы
- `task3_http_file/server.py`
- `task3_http_file/index.html`

### Запуск
```bash
python task3_http_file/server.py
```

Открыть в браузере:
- `http://127.0.0.1:9003`

### Результат
В браузере открывается HTML-страница из `index.html`.

# Задание 4 — Многопользовательский чат (TCP + threading)

**Описание:** реализован многопользовательский чат. Сервер принимает несколько клиентов.  
Для каждого клиента создаётся отдельный поток, сообщения пересылаются всем подключённым.

### Файлы
- `task4_chat/server.py`
- `task4_chat/client.py`

### Запуск
Сервер (окно 1):
```bash
python task4_chat/server.py
```

Клиенты (несколько окон терминала):
```bash
python task4_chat/client.py
```

### Результат
Сообщение, отправленное в одном клиенте, отображается у других клиентов.

# Задание 5 — Простой веб-сервер GET/POST (журнал оценок)

**Описание:** написать простой HTTP-сервер, который:
1) принимает и записывает дисциплину и оценку (POST);  
2) отдаёт HTML-страницу со всеми сохранёнными оценками (GET).

### Файлы
- `task5_http_journal/server.py`

### Запуск
```bash
python task5_http_journal/server.py
```

Открыть в браузере:
- `http://127.0.0.1:9005`

### Результат
- при `GET /` отображается список дисциплин и оценок + форма добавления;
- при отправке формы (POST) запись добавляется и появляется в списке.
