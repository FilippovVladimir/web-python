# Лабораторная работа 4

## Тема
Реализация клиентской части средствами Vue.js.

## Предметная область
**Авиарейсы**

## Цель работы
Создать клиентское приложение на Vue.js, которое взаимодействует с backend API из лабораторной работы 3.

## Используемые технологии
- Vue 3
- Vite
- Axios
- Vue Router
- Bootstrap 5
- Django REST Framework backend
- Djoser token auth

## Настройка взаимодействия с backend
Frontend работает на:
- `http://localhost:5173`

Backend работает на:
- `http://127.0.0.1:8000`

Для взаимодействия с разных origin на backend была настроена поддержка CORS через `django-cors-headers`.

## Реализованные интерфейсы

### Страница "Рейсы"
- отображение списка рейсов
- поиск по рейсам
- кнопка создания брони для авторизованного пользователя

### Страница "Регистрация"
- регистрация нового пользователя через Djoser

### Страница "Вход"
- получение токена
- сохранение токена в `localStorage`

### Страница "Мои брони"
- просмотр собственных бронирований
- удаление брони

### Страница "Отзывы"
- просмотр отзывов
- добавление нового отзыва
- удаление отзыва

### Страница "Аккаунт"
- получение информации о текущем пользователе через `/api/auth/users/me/`
- изменение username и email

## Авторизация на frontend
После входа токен сохраняется в браузере:
```text
localStorage['auth_token']
```

Во все запросы автоматически добавляется заголовок:
```text
Authorization: Token <token>
```

## Маршрутизация
Использованы маршруты Vue Router:
- `/` - рейсы
- `/login` - вход
- `/register` - регистрация
- `/reservations` - мои брони
- `/reviews` - отзывы
- `/account` - аккаунт

## Взаимодействие с API
Для запросов к backend используется Axios.  
Основные методы:
- получение списка рейсов
- регистрация и вход
- получение текущего пользователя
- создание и удаление бронирований
- создание и удаление отзывов

## Интерфейс
Для простого и понятного пользовательского интерфейса использован Bootstrap 5 как аналог библиотеки компонентов.

## Вывод
В ходе работы была реализована клиентская часть приложения на Vue.js, настроено взаимодействие с backend API, реализованы интерфейсы регистрации, входа, работы с рейсами, бронированиями, отзывами и аккаунтом пользователя.
