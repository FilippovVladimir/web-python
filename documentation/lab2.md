# Лабораторная работа 2

## Тема
Реализация простого сайта средствами Django.

## Предметная область
**Табло отображения информации об авиаперелетах**

## Цель работы
Освоить создание веб-приложения на Django, работу с моделями, шаблонами, представлениями, регистрацией пользователей и административной панелью.

## Описание предметной области
Система хранит информацию о рейсах:
- номер рейса
- авиакомпания
- город вылета
- город прилета
- время вылета и прилета
- тип рейса
- номер гейта

Пользователи могут:
- регистрироваться
- просматривать рейсы
- бронировать места на рейсах
- редактировать и удалять свои бронирования
- оставлять отзывы о рейсах

Администратор может:
- управлять рейсами
- управлять бронированиями
- вписывать номер билета в бронирование через Django admin

## Модель данных

### Flight
Хранит информацию о рейсе:
- `flight_number`
- `airline`
- `departure_city`
- `arrival_city`
- `departure_time`
- `arrival_time`
- `flight_type`
- `gate`

### Reservation
Хранит бронирование места:
- пользователь
- рейс
- место
- дата создания
- номер билета

### Review
Хранит отзыв:
- рейс
- автор
- дата рейса
- текст
- рейтинг

## Реализованный функционал

### Регистрация пользователей
Реализована форма регистрации на основе `UserCreationForm`.

### Просмотр рейсов
На главной странице выводится список рейсов с краткой информацией.

### Бронирование мест
Авторизованный пользователь может создать бронирование на рейс.

### Редактирование и удаление своих бронирований
Пользователь может изменять и удалять только свои собственные бронирования.

### Отзывы
Авторизованные пользователи могут оставлять отзывы к рейсам.

### Административная панель
Через Django admin администратор может:
- создавать и редактировать рейсы
- просматривать бронирования
- вписывать номер билета в бронирование
- просматривать отзывы

### Таблица пассажиров рейса
На странице рейса формируется таблица пассажиров:
- пользователь
- место
- номер билета

## Дополнительный функционал

### Bootstrap меню
На сайте реализовано навигационное меню на Bootstrap.

### Пагинация
Список рейсов и список бронирований разбиваются на страницы.

### Поиск
Реализован поиск по рейсам и по бронированиям.

В ходе работы был реализован полноценный Django сайт по выбранной предметной области. Освоены модели, представления, шаблоны, формы, авторизация, Django admin, пагинация и поиск.
